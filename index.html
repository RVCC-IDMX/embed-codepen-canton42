Pen: 3D CSS Typography

by Noah Blon

In this pen the author has created animated 3D text using only CSS and divs! The code uses a combination of transform and translation properties to achieve the effect. This pen shows that CSS is more than capable of rendering 3D imagery as well as normal text and colors.
